# RetailFlow

RetailFlow is an Excel-based departmental store analytics project.

## How to use
1. Fill the CSV template in Excel.
2. Save it as `.csv` or `.xlsx`.
3. Open `retailflow.html`.
4. Login using:
   - manager@retailflow.com
   - mca123
5. Upload the file and view analytics.

## Required columns
- date
- department
- product
- units
- unit_price
- cost_per_unit
- payment_mode
- stock_left
- remarks
